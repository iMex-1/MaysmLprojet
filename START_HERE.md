## 🚀 Installation en 3 Commandes

```bash
# 1. Installer Breeze et dépendances
composer require laravel/breeze --dev && php artisan breeze:install blade && npm install && npm run build

# 2. Base de données
php artisan migrate:fresh --seed

# 3. Démarrer
php artisan serve
```
